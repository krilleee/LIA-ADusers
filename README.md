# Getting Started

1. Clone the repository
2. Create a new branch for your work
3. Check the diagrams and write a script for the purpose
4. Commit your changes
5. Open a Pull Request

```bash
git clone git@github.com:krilleee/LIA-ADusers.git
cd LIA-ADusers
git checkout -b feature/new-user-script
